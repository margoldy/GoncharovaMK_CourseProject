# Семейный бюджет — веб-приложение для учёта финансов

## О проекте

**Семейный бюджет** — это веб-приложение для учёта доходов и расходов семьи. Оно позволяет нескольким членам семьи вести общий бюджет, отслеживать траты, планировать расходы и анализировать финансовое состояние.

Проект разработан в рамках курсовой работы по дисциплине «Создание программного обеспечения» (РТУ МИРЭА, 4 семестр).

## Основные возможности

- **Регистрация и вход** — пользователи регистрируются, после чего администратор подтверждает учётную запись
- **Разграничение ролей** — администратор управляет пользователями, обычные пользователи работают только со своими операциями
- **Управление доходами** — добавление доходов с автоматическим распределением по трём счетам (70% на общий, 20% на накопления, 10% в личную заначку)
- **Управление расходами** — добавление фактических и плановых расходов с проверкой достаточности средств
- **Переводы между счетами** — перемещение средств между общим счётом, накоплениями и заначкой
- **Финансовые отчёты** — формирование отчёта за выбранный период с возможностью экспорта в Excel
- **Журнал действий** — аудит всех значимых операций пользователей

## Установка и запуск
### Требования
- Node.js (версия 18 или выше)
- npm (версия 9 или выше)
- Браузер (Chrome, Firefox, Edge)

**Шаг 1. Клонирование репозитория**
```bash
git clone <ссылка_на_репозиторий>
cd family-budget
```
**Шаг 2. Настройка и запуск бэкенда**
```bash
cd backend
npm install
```
Создать файл `.env` в папке `backend` со следующим содержимым:
```
PORT=5000
JWT_SECRET=your_secret_key_here
```
> **Примечание:** `JWT_SECRET` должен быть уникальной строкой.

Запуск сервера:
```bash
node server.js
```
При успешном запуске в консоли появится сообщение:
```
База данных инициализирована
Сервер запущен на http://localhost:5000
```
**Шаг 3. Настройка и запуск фронтенда**
Открыть новый терминал и выполнить:
```bash
cd frontend
npm install
npm run dev
```
При успешном запуске в консоли появится сообщение:
```
VITE v5.x.x ready in xxx ms
➜ Local: http://localhost:5173/
```
**Шаг 4. Открытие приложения в браузере**
Перейти по адресу: `http://localhost:5173`

**Шаг 5. Вход в систему**
**Тестовый вход:**  
Логин: `admin`  
Пароль: `admin123`

### Альтернативный запуск после установки (одной командой)
```bash
cd ..
npm start
```
